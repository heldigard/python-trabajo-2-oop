# Trabajo 2: Programación Orientada a Objetos

Objetivo:
Diseñar clases para modelar un sistema de gestión de estudiantes y sus notas.

Requisitos:
- Clase `Estudiante` con atributos `nombre`, `apellidos`, `id` y `notas` y métodos para añadir nota y calcular promedio.
- Clase `Aula` o `Curso` que gestione varios estudiantes y calcule estadísticas agregadas.
- Archivo `main.py` con ejemplos de uso y `tests/` con pytest.

Entrega:
- `estudiante.py`, `curso.py`, `main.py`, `tests/`.
